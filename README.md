# 🎓 Student Management System (Java)

## 📌 Project Overview
The Student Management System is a simple yet effective Java-based console application designed to manage student records in an organized and efficient way. This project replaces traditional manual record-keeping methods with a digital solution that is faster, more reliable, and easier to use.

The system allows users to perform basic operations such as adding new student records, viewing existing records, and deleting records when required. It is built using core Java concepts and demonstrates the practical use of object-oriented programming.

---

## ❗ Problem Statement
Managing student data manually using registers or paper-based systems can be time-consuming, error-prone, and inefficient. It becomes difficult to store, retrieve, and update records as the number of students increases.

This project aims to solve these problems by providing a simple digital system that ensures:
- Faster data handling  
- Reduced human errors  
- Easy data management  

---

## 🧑‍💻 Implementation Details
The system is implemented using Java and follows a menu-driven approach. It uses object-oriented programming concepts such as classes and objects.

- A `Student` class is used to represent student data (ID, name, age)  
- The `Main` class handles user interaction and program execution  
- An `ArrayList` is used to store student records dynamically  
- The system continuously runs using a loop until the user chooses to exit
